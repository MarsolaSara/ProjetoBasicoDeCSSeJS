# ✨ Premium Landing Page

Uma landing page moderna e responsiva desenvolvida com **HTML, CSS e JavaScript puro**, focada em experiência do usuário, animações suaves e design elegante.

O projeto simula a página inicial de uma empresa digital, apresentando serviços, informações institucionais e elementos visuais interativos.

## 🎨 Recursos implementados

### Interface moderna

O projeto utiliza:

- Gradientes.
- Sombras e transições.
- Componentes reutilizáveis.
- Tipografia limpa.
- Layout responsivo.

### Animações

Foram utilizadas animações com:

- `IntersectionObserver`
- `transform`
- `transition`
- `@keyframes`

Essas técnicas permitem uma experiência mais dinâmica e agradável para o usuário.
